MeetingSpot

アプリ概要

MeetingSpotは、複数人での待ち合わせ場所選びをAIがサポートするアプリです。
「どこで待ち合わせしよう?」という悩みを、AIが解決します!

こんな経験ありませんか?
・友達3人で集まりたいけど、誰の家からも近い場所がわからない
・待ち合わせ場所を決めるのに時間がかかってしまう
・結局誰かが不便な場所になってしまう

MeetingSpotなら、全員が納得できる場所をAIが提案!
各自の出発地点と待ち合わせ時間を入力するだけで、全員にとって公平で便利な場所を3つ提案します。

🆕 チャット機能で細かい要望にも対応!

AIが提案した場所に対して、「もっと静かな場所がいい」「カフェがある場所がいい」などの追加要望をチャットで伝えられます。
納得できる待ち合わせ場所が見つかるまで、何度でも相談できます!

特長

  • 2人から複数人対応 - 友達との待ち合わせから、グループでの集まりまで
  • 全員に公平 - みんなの移動時間を考慮して、誰にとっても便利な場所を提案
  • 時間帯を考慮 - 待ち合わせ時間に開いているお店や場所を優先的に提案
  • チャットで細かく調整 - 提案に満足できなければ、追加の要望をチャットで伝えられる
  • シンプルで簡単 - 出発地点と時間を入力するだけ。難しい操作は一切なし
  • プライバシー重視 - 位置情報は検索時のみ使用。個人情報は一切保存しません

使い方

3ステップで完了!

  1. 出発地点を入力
    参加者全員の出発地点(駅名や住所)を入力します。最低2人から、何人でもOK!
  2. 待ち合わせ時間を設定
    カレンダーから日時を選ぶだけ。今日でも、来週でも指定できます。
  3. 「待ち合わせ場所を探す」をタップ
    AIが3つの候補を提案します!

気に入らなければ...

  • 💬 チャットで要望を伝える
    「もっと静かな場所がいい」「カフェがある場所がいい」など、具体的な要望を入力すると、AIが再提案してくれます。
  • 🔄 「探す」ボタンで再検索
    別の候補が欲しい時は、何度でも検索できます。

場所が決まったら

提案された場所へのルートは、GoogleマップやYahoo!マップなど、お使いの地図アプリで確認してください。

こんな時に便利!

  • 👥 友達との待ち合わせ - 3人以上で集まる時、誰の負担も少ない場所を探せます
  • 💼 ビジネスでの打ち合わせ - 取引先との中間地点を素早く見つけられます

  • 💑 デートや家族での外出 - お互いに便利な場所で待ち合わせできます

詳しい使い方

1. 出発地点の入力

  • 最低2人から入力できます
  • 「＋参加者を追加」ボタンで簡単に人数を増やせます
  • 不要な参加者は「−」ボタンで削除できます(最低2人は必須)
  • 「リセット」ボタンで全て初期状態に戻せます
  • 駅名、住所、ランドマーク名など、わかりやすい場所を入力してください

2. 待ち合わせ時間の設定

  • 日付と時間を選択できます
  • 現在時刻より未来の時間を指定してください
  • 時間帯によって、開いているお店を優先的に提案します

3. 検索と提案

  • 「待ち合わせ場所を探す」ボタンをタップ
  • AIが最適な場所を検索中...と表示されます
  • 3つの候補が表示されます
  • 各候補には、場所の特徴やアクセス方法が詳しく説明されます

4. チャット機能で細かく調整

  • 提案された場所に満足できない場合、チャット欄に要望を入力
  • 例:「もっと静かな場所」「カフェがある場所」「駅から近い場所」
  • AIが要望を理解して、新しい候補を提案します
  • 何度でも質問できます!

5. 場所が決まったら

  • 決定した場所を地図アプリで検索
  • 各自のルートを確認して、待ち合わせ!
