MeetingSpot

アプリ概要

MeetingSpotは、AIが提案する皆の待ち合わせ場所検索アプリ
待ち合わせ場所で悩む時間を減らすためのAIを活用します

複数の出発地点から最適な待ち合わせ場所を検索し、3つの候補をAIが提案します。
AIが選ぶベストな待ち合わせ場所で、楽しい時間を！

提案された場所へのルートは、普段お使いの地図アプリやウェブサービスでご確認下さい。

特長

  • 2人から複数人対応の待ち合わせ場所検索
  • 全員の移動時間を考慮した公平な場所選び
  • 時間帯に応じた営業中の場所を優先提案
  • シンプルで直感的な操作性
  • プライバシーに配慮（位置情報は検索時のみ使用。個人情報の保存なし）

アプリの使い方

簡単な使用方法

  1. 参加者の出発地点を入力
  2. 待ち合わせ時間を設定
  3. 「待ち合わせ場所を探す」ボタンをタップ
  4. 提案された場所が気に入らない場合は納得するまで、「探す」ボタンを押す
  5. 待ち合わせ場所を決めたら、そこまでのルートは、普段お使いの地図アプリやウェブサービスでご確認下さい。

アプリの使い方（詳細）

  1. 参加者の出発地点を入力
    • 最低2名から入力可能
    • 必要に応じて参加者を追加（＋ボタンで簡単追加）
    • 不要な参加者は削除可能（最低2名は必須）
    • リセットボタンで初期画面
  2. 待ち合わせ時間を設定
    • 日付と時間をホイール選択で簡単設定
    • 現在時刻から未来の時間を指定可能
  3. 「待ち合わせ場所を探す」ボタンをタップ
    • AIが最適な待ち合わせ場所を検索
    • 処理中はローディング表示で状態を確認可能
  4. 提案された場所を決める
    • カフェや駅などの候補から最適な場所を選択
    • 納得するまで、「探す」ボタンを押す
  5. 待ち合わせ場所を決めたら、そこまでのルートは、普段お使いの地図アプリやウェブサービスでご確認下さい。
